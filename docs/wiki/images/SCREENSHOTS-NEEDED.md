## How to Take Screenshots

1. Run the TLC dashboard or command
2. Use your OS screenshot tool
3. Crop to the relevant area
4. Save as PNG with descriptive name
5. Optimize file size (aim for <100KB per image)

## Naming Convention

```
{feature}-{context}.png
```

Examples:
- `dashboard-tasks.png`
- `build-parallel.png`
- `verify-checklist.png`
